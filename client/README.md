# Client Readme
# Fitness Habit Tracker Client

Modern React frontend for the Fitness Habit Tracker application.

## Setup

1. **Install dependencies**:
```bash
npm install
```

2. **Start development server**:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
npm run build
```

## Features

- 🔐 User Authentication (Register/Login)
- 📊 Habit Dashboard
- ✓ Mark habits as complete
- 🔥 Streak tracking
- 📈 Progress statistics
- 🎨 Colorful habit cards
- 🗑️ Delete habits
- 📱 Responsive design
